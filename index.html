5c Stability Check in Wind Condition (Unanchored Tank — §5.11)

Three conditions per API-650 §5.11.2 must all be satisfied for an unanchored tank:
(1) 0.6·Mw + Mpi < MDL/1.5 + MDLR  ·  (2) Mw + Fp·Mpi < (MDL+MF)/2 + MDLR  ·  (3) Mws + Fp·Mpi < MDL/1.5 + MDLR
If any condition fails, anchorage is required. Mpi uses design internal pressure; Mpop (operating, with Fp=1) is used in conditions 2 & 3 per the source workbook.

10 Anchorage Inputs

10b Bolt Properties & Sizing

Anchor bolt root area is taken from the standard metric bolt table (M10–M100), reduced by a 6 mm corrosion allowance on the root diameter. Required bolt count N = Umax / (Acorr · fallow); spacing = π·BCD / N. Bolt yield is capped at 380 MPa per API-650 Table 5.20a.
